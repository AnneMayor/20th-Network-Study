- 분량은 매주 두 장씩. 기획 나와서 프로젝트 시작할 적에는 1장씩 조율하는 방향으로! 예외적으로, 첫 째주는 1장만~
- 각자 읽어오고 기술적인 토의(질답)하면서 해당 내용을 깃헙에 .md 파일로 올리기
- 파일에 올릴 내용은 해당 스터디 시간에 나눴던 기술적인 내용과 질문에 대해 정리하기(정리 순서: @이다혜 → @김수빈 -> @이우성 -> @김유비)
